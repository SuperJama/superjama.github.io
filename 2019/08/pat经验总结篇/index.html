1 C++/C的一些需要注意的地方

1.1 变量

变量的定义：
变量类型 变量名; /// 可以同时初始化

变量类型：
int         取值范围：-2^31~2^31-1 (-2*10^9~2*10^9);   在scanf()和printf()中的格式：%d;    ///其中%?md可以确定最少输出m位，?为填充
long long   取值范围：-2^63~2^63-1 (-2*10^18~2*10^18); 在scanf()和printf()中的格式：%lld   ///%?md用法同上
float       取值范围：-2^128~2^128 (实际精度6～7位);     在scanf()和printf()中的格式：%f     ///%.mf可以确定保留小数点后m位(不是四舍五入)
double      取值范围：-2^1024~2^1024 (实际精度15～16位); 在scanf()和printf()中的格式：%lf   ///%.mld用法同上
char        取值范围：-128~127     ();                 在scanf()和printf()中的格式：%c
bool        取值范围：0 or 1       (false true);       ----
char[]      ---	                  ---                在scanf()和printf()中的格式：%s

数组变量定义：
变量类型 变量名[]; /// 可以同时初始化

输入与输出：
ep：scanf("%d", &a); /// &符不可少，但可在数组名前可省略
可在""中自定输入格式
需注意：%c可以读入空格；%s可以空格结束

ep：printf("%d", a);
需注意：转义字符\

ep：char a = getchar(); ///读入一个字符('\n', ' '皆能读入)
需注意：在使用getchar()读入字符串时，需加'\0'作为结尾

ep：putchar(a); /// 输入a

ep：gets(str); ///读入字符串(以'\n'结束)

ep：puts(str); /// 输出字符串

赋值memeset()：
ep：memeset(a, 0, sizeof(a)); /// 只建议使用此法赋初值 0 / -1

赋值 fill():
ep：fill(a, a + 10, 2); /// 将a[10]中所有元素赋值为‘2’

1.2 浮点数的比较(考虑误差)

由于计算机中采用有限位的二进制编码，因此浮点数在计算机中的存储并不总是精确的。 为了将这极小的误差消除，我们可以引入一个极小数eps(一般取1e-8较为合适)。

设定义两个变量a，b:
等于操作：a - b < eps;
大于操作：a - b > eps;
小于操作：a - b < -eps;
大于等于操作：a - b > -eps;
小于等于操作：a - b < eps;
