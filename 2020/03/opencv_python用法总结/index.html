Opencv_python用法总结

三月 27, 2020 发布在 Python

Opencv_python的一些简单的理解与介绍。

目录

  • 函数
      • cv2.imread(arg1， arg2)
      • cv2.imshow(arg1， arg2)
      • cv2.waitKey()和cv2.destroyAllWindows()

函数

cv2.imread(arg1， arg2)

  • 该函数的功能是读取给定位置的图片
  • 该函数有两个参数， path 和 标记位 path即为图片的路径，标记位从 -1～3 CV_LOAD_IMAGE_UNCHANGED – 在每个通道中，每个像素的位深为8 bit，通道数（颜色）保持不变。 CV_LOAD_IMAGE_GRAYSCALE – 位深=8 bit 通道数=1（颜色变灰） CV_LOAD_IMAGE_COLOR -位深=?, 通道数=3 CV_LOAD_IMAGE_ANYDEPTH – 位深不变 ,通道数=? CV_LOAD_IMAGE_ANYCOLOR – 位深=?, 通道数不变

例如：

path = "/aaa/bbb.jpg"
img = cv2.imread(path, 1)

此时获得的img是一个<class 'numpy.ndarray'>类型变量，一般有三个维度，分别是图像的高（h）、宽（w）和通道数。通道的排列是BGR，通过切片可以加深理解。

cv2.imshow(arg1， arg2)

  • 用来显示图像，窗口会自动调整为图像大小。第一个参数是窗口的名字，然后是图像。

例如：

#接上面
cv2.imshow("图片", img)

cv2.waitKey()和cv2.destroyAllWindows()

  • cv2.waitKey() 等待任意键盘输入。
  • cv2.destroyAllWindows() 销毁所创建的所有窗口。

例如：

#接上面
cv2.waitKey()
cv2.destroyAllWindows()
