基于差值的SRR方法

最近邻差值（Nearest Neighbor Interpolation）

该方法指重建后的HR图像的每个像素点由距离其最近的原像素点决定。 公式如下（以2倍数放大举例）：

$$ d = min \ \left\{ \begin{array}{lr} d[(u, v)(2i, 2j)], d[(u, v),(2i, 2j + 2)], \\ d[(u, v)(2i+2, 2j)], d[(u, v),(2i+2, 2j + 2)] \end{array} \right\}. $$

（u,v）为扩充后的HR图像像素点，（i,j）为原始LR图像像素。 实现过程：

def NearestNeighborInterpolation(img, times=2):
    #获取图像高、宽
    h, w = img.shape[0], img.shape[1]
    #计算新的高、宽
    new_h, new_w = int(times * h), int(times * w)
    #生成新图像
    ret_img = np.zeros(shape=(new_h, new_w, 3), dtype='uint8')
    #为新图像添加像素信息
    for i in range(new_h):
        for j in range(new_w):
            res = minDistance(i, j, times, new_h, new_w)
            ret_img[i, j] = img[res[0], res[1]]
    return ret_img

'''用来获取距当前像素点最近的原图像像素点'''
def minDistance(h, w, times, new_h, new_w):
    #计算新图与原图比值
    s_h, s_w = 1.0/ times, 1.0 / times
    #获取当前像素点左上角点原图像素点位置
    o_h, o_w = int(s_h * h), int(s_w * w)

    #计算距离
    dist = []
    dist.append(((times*o_h - h) ** 2 + (times*o_w - w) ** 2))
    dist.append(((times*o_h - h) ** 2 + (times*(o_w + 1) - w) ** 2))
    dist.append(((times*(o_h + 1) - h) ** 2 + (o_w - w) ** 2))
    dist.append(((times*(o_h + 1) - h) ** 2 + (times*(o_w + 1) - w) ** 2))
    min = 0
    for i in range(4):
        if dist[min] >= dist[i]:
            min = i
    if min == 0:
        return (o_h, o_w)
    if min == 1:
        if  o_w + 1 >= new_w / times:
            return (o_h, o_w)
        return (o_h, o_w+1)
    if min == 2:
        if o_h + 1 >= new_h / times:
            return (o_h, o_w)
        return (o_h+1, o_w)
    if min == 3:
        if o_h + 1 >= new_h / times or o_w + 1 >= new_w / times:
            return (o_h, o_w)
        return (o_h+1, o_w+1)

双线性插值法

双线性插值是两个linear插值组合使用的形式，其下图是其原理： 假设P点为要求像素的点，Q11 = (x1, y1)、Q12 = (x1, y2), Q21 = (x2, y1) 以及 Q22 = (x2, y2)分别是它旁边的四个点，想要求P点的像素，需先求R1和R2的像素（第一次linear插值），之后根据R1，R2求得P点的像素。公式如下：

$$ f(R_1) = {x_2 - x \over x_2 - x_1} * f(Q_{11}) + {x - x_1 \over x_2 - x_1} * f(Q_{21}) \\ f(R_2) = {x_2 - x \over x_2 - x_1} * f(Q_{12}) + {x - x_1 \over x_2 - x_1} * f(Q_{22}) \\ f(P) = {y_2 - y \over y_2 - y_1} * f(R_1) + {y - y_1 \over y_2 - y_1} * f(R_2) $$

由于图像双线性插值只会用相邻的4个点，因此上述公式的分母都是1。 设P = （x, y）其在原图像的位置为（x / times, y / times）。设其左上角原图像像素为（i ,j），则P点在原图像的位置为（i + u, j + v），i、j为整数，u、v为小数。 上述公式可用矩阵表示：

$$ f(P) = [1-v, v] \left[\begin{matrix} f(Q{11}) & f(Q{21}) \ f(Q{12}) & f(Q{22}) \end{matrix}\right] [1-u, u]^T $$

双三次插值（BiCubic插值）

即插值点的像素由其周围16个点的像素值共同决定。 Bicubic函数公式为：

$$ S(x) = \begin{cases} (a+2)|x|^3-(a+3)|x|^2+1, & 0\le|x|\le1 \ a|x|^3-5a|x|^2+8a|x|-4a, & 1\le|x|\le2 \ 0, & other \end{cases} $$

本算法取a=-0.5效果最佳。 点（x, y）像素值可由下面公式获得：

$$ f(x, y) = f(i+u, j+v) = A*B*C
A= [\begin{matrix} S(u+1) & S(u) & S(u-1) & S(u-2) \end{matrix}]
B= \left[\begin{matrix} f(i-1, j-1)&f(i-1,j)&f(i−1, j+1)&f (i−1, j+2)
f(i,j−1)&f(i,j)&f(i,j+1)&f(i,j+2)
f(i+1, j−1) &f(i+1, j) &f(i+1, j+1) &f(i+1, j+2)
f(i+2, j−1)&f(i+2, j) &f(i+2, j+1) &f(i+2, j+2) \end{matrix}\right]
C=[\begin{matrix} S(v+1) & S(v) & S(v-1) & S(v-2) \end{matrix}]^T $$

(x, y)是缩放后的图像的像素点位置，将其除以缩放倍数即可得到(i+u, j+v)。 坐标i、j为非负整数，u、v是浮点数，取值在[0,1)之间。 示意图：
